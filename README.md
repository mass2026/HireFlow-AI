# 🚀 HireFlow-AI

> Smarter Hiring. Faster Decisions.

An enterprise-grade AI-powered recruitment platform built with Java, Spring Boot, React, PostgreSQL, JWT, Docker, and modern software engineering practices.

---

## 🌟 Features

- Secure Authentication (JWT)
- Role-Based Access Control
- Candidate Management
- Recruiter Dashboard
- Company Management
- Job Posting
- Job Application Tracking
- Resume Upload
- Interview Scheduling
- AI Resume Analysis (Upcoming)

---
