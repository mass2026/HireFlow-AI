
# Jobs

Purpose:
Stores all job postings created by recruiters.

Relationships:
- One Company → Many Jobs
- One Recruiter → Many Jobs
- One Job → Many Applications

Important Fields:
- job_title
- description
- employment_type
- experience_min
- experience_max
- salary_min
- salary_max
- work_mode
- city
- state
- country
- department
- interview_mode
- application_deadline
- job_status
